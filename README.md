# An introduction into ES2015+

Materials for the lecture.

## Requirements

* Node &ge; 8.0
* `[optional]` Visual Studio Code &ge; 1.14

## Instructions

* No setup required
* Just run a file with `Node` in console
* To debug with `VSCode` set a brakepoint and press `F5`
